# Smart Table

This repository contains source code for Smart Table firmware.

## Prerequisites

-   ESP-IDF 4.4 or 4.5

## How to build

```
    idf.py build
```

## Flashing

```
    idf.py flash -p <PORT>
```

## References

-   ESP-IDF 4.4 documentation: https://docs.espressif.com/projects/esp-idf/en/release-v4.4/esp32/get-started/index.html
